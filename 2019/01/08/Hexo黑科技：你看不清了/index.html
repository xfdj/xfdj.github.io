Hexo黑科技：你看不清了

发表于 2019-01-08 更新于 2021-03-15 本文字数： 809 阅读时长 ≈ 1 分钟

偶然在一个博客发现Hexo的黑科技，据说你 撸多了 会视物模糊？

怎么做到的？

实际上这是Hexo的一个有趣的插件实现的。模糊的文字点击即可查看，再次点击则又会模糊。首先来安装插件。

安装

在博客目录下输入

yarn add hexo-spoiler

或

npm install hexo-spoiler --save

完成安装。

使用

在你写博客的时候，如果想马赛克某段文字，你可以这么使用。

{% spoiler 没错，你眼睛花了。 %}

效果嘛就是这样的：

没错，你眼睛花了。

什么地方可以用

  • 正常文本： 可以遮蔽

  • 带格式的文本：

    • 斜体： 可以遮蔽

    • 粗体： 可以遮蔽

    • 下标： 可以遮蔽

    • 删除线： 可以遮蔽

    • 下划线： 可以遮蔽

    • 行内代码：{% spoiler 可以遮蔽 %}

    • 超链接： 可以遮蔽，不过点我会被驱逐至百度

    • 引用：

      可以遮蔽

带格式的文本需要把格式符放在外面，像这样：

**{% spoiler 把粗体格式的符号放在外面 %}**
[{% spoiler 把超链接的符号放在外面 %}](https://www.baidu.com)

不知道是不是markdown渲染引擎的缘故，如果格式符放在里面是没有格式效果的。另外，行内公式还无法使用马赛克。

就是这么简单~目前除了好玩之外也想不到有什么用处了，也许以后可以在出问题的时候把答案遮住？嗯，似乎是个不错的主意。
