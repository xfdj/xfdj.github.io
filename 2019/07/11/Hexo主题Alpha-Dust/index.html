Hexo主题Alpha Dust

发表于 2019-07-11 更新于 2021-09-03 本文字数： 2.1k 阅读时长 ≈ 4 分钟

最近浏览博客的时候发现了一个很有科技感的博客，拉到网页底部一看还是一个Hexo的博客，于是我搜索了一下这个博客网站的主题，也就是这篇文章要介绍的Hexo主题：Alpha Dust。

作者

这个主题的作者名叫Jonathan Klughertz，是一个全栈工程师。除了Alpha Dust以外，他还制作了很多Hexo的主题，这都可以在他的博客上找到。（点此访问Jonathan的博客）

主题概览

这个主题的博客介绍和Github项目链接下面给出：

  • 博客：New Hexo Theme Alpha Dust - CodeBlocQ
  • Github：klugjo/hexo-theme-alpha-dust: 🌠 Original Futuristic Hexo Theme

网站的demo可以访问下面的链接：

  • Alpha Dust

安装

和安装NexT时候类似，都是直接把Github上的项目克隆到本地。

在博客根目录下执行下面的指令：

git clone https://github.com/klugjo/hexo-theme-alpha-dust themes/alpha-dust

然后更新站点配置文件_config.yml来将主题设为Alpha Dust。如下所示：

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
# theme: next
theme: alpha-dust

重新部署一下博客，就能看到这个新的主题了。

常用配置

配置也是和其它主题一样靠编辑主题配置文件themes/alpha-dust/_config.yml实现。

Menu

# Header
menu:
  Home: /
  Archives: /archives
  # About: /about.html
  Tags: /tags
  Categories: /categories
  # Contact: /contact.html

页脚

页脚的关于和版权部分的文字可以通过下面的片段来编辑：

# Footer About Text
footer_about: "This theme was developed by <a href=\"https://github.com/klugjo\">Jonathan Klughertz</a>. The source code is available on Github. Create Websites. Make Magic."

# Footer Copyright Line
footer_copyright: "@Untitled. All right reserved | Design & Hexo <a href=\"http://www.codeblocq.com/\">Jonathan Klughertz</a>"

对应的是页脚About部分的显示：

社交账户图标

# Social Accounts
twitter_url: https://twitter.com/?lang=en
facebook_url: https://www.facebook.com/
instagram_url: https://www.instagram.com/
dribble_url: https://dribbble.com/
github_url: https://github.com/klugjo/hexo-theme-alpha-dust
googleplus_url: https://plus.google.com/
behance_url: https://www.behance.net/
fivehundredpx_url: https://500px.com/
email_url: test@example.com
rss_url: \#

这会更改同样在页脚的社交图标的显示和链接，如果没有链接就不显示。

评价

由于这是个人作品，从主题配置文件就可以看得出功能偏少，另外，网页的排版还不够灵活，比如全屏网页时文章左右没有留白，这样看起来不是很舒服，还有首页上的文章都是全文显示，不方便浏览，代码框没有复制按钮等等，就高级功能来说，NexT还是更胜一筹，所以玩了之后，我还是打算换回NexT。如果需要实现这些功能需要自己进一步改进主题，目前自己暂时没有这个能力。不过这个主题还是很好看的，期待作者后续能够给主题加入更多功能。
