Ubuntu如何安装tar.gz版的Flash Player

发表于 2019-02-16 更新于 2021-09-03 本文字数： 446 阅读时长 ≈ 1 分钟

在Ubuntu 18.04里打开Firefox，发现视频看不了，需要安装Flash Player，但是在Flash Player的下载页面下载时出现了问题，弹出一个错误提示框，显示“未知频道‘bionic-partner’。所幸Flash Player提供了tar.gz的安装方式。

  1. 将tar.gz安装包下载下来解压。

  2. 将libflashplayer.so文件拷贝到Firefox的plugins目录，我的电脑Firefox的plugins目录是：/usr/lib/firefox-addons/plugins。如果通过界面操作无法操作，可以使用命令行：

    sudo cp libflashlayer.so /usr/lib/firefox-addons/plugins

  3. 将安装包的usr文件夹下的所有文件拷贝到/usr目录下，可以通过命令：

    sudo cp -r usr/* /usr

  4. 重新打开Firefox浏览器，Flash player就可以使用了。
