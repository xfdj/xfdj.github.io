最小费用最大流简介

发表于 2021-07-11 更新于 2021-11-06 本文字数： 3.1k 阅读时长 ≈ 5 分钟

任何容量网络的最大流的流量是唯一确定的，但是可行的最大流不唯一。

假设一个容量网络\(G=(V, E)\)。源点\(V_s\)，汇点\(V_t\)，弧\(<u, v> \in E\)。\(c(u, v)\)表示弧\(<u, v>\)的容量，\(r(u, v) \geq 0\)表示弧\(<u, v>\)的单位流量费用。要求一个最大流\(f\)，使得\(f\)的总费用\(r(f)=\sum_{<u, v> \in E}f(u, v) \times r(u, v)\)最少，称为最小费用最大流。

寻找最大流的方法是从某个可行流\(f\)出发，找到关于\(f\)的一条增广路\(P\)，沿着\(P\)调整\(f\)，如此反复直到不存在增广路为止。

现在要寻找最小费用最大流，首先考察当沿着一条增广路\(P\)，以改进量\(\alpha=1\)调整可行流\(f\)，得到新的可行流\(f'\)（显然\(|f'|=|f|+1\)）后，\(r(f')\)比\(r(f)\)增加了多少？ \[ \begin{aligned} r\left(f'\right)-r(f) &= \sum_{<u, v> \in E} f'(u, v) * r(u, v) - \sum_{<u, v> \in E} f(u, v) * r(u, v) \\ &= \sum_{<u, v> \in E} (f'(u, v) - f(u, v)) * r(u, v) \\ &= \left[\sum_{e \in P+} r(e) *\left(f'(e)-f(e)\right)-\sum_{e \in P-} r(e) *\left(f'(e)-f(e)\right)\right] \\ &= \sum_{e \in P+} r(e)-\sum_{e \in P-} r(e) \end{aligned} \] 把\(\sum_{e \in P+} r(e)-\sum_{e \in P-} r(e)\)叫做这条增广路\(P\)的费用。

显然，如果\(f\)是流量为\(|f|\)时的最小费用流，\(P\)是关于\(f\)费用最小的增广路，沿\(P\)调整\(f\)得到\(f'\)，就是流量为\(|f'|\)时的最小费用流。

寻找最小费用最大流的问题就变成了，从\(f=\{0\}\)开始增广，如何寻找最小费用增广路？

构造容量网络关于\(f\)的伴随网络\(W(f)\)：顶点是源网络中的顶点，每条弧\(<u, v>\)都变成两个相反的弧\(<u, v>\)和\(<v, u>\)，弧的权值为： \[ W(u, v) = \begin{cases} r(u, v) & \text{若} f(u, v) < c(u, v) \\ +\infty & \text{若} f(u, v) = c(u, v) \end{cases} \] \[ W(v, u) = \begin{cases} -r(u, v) & \text{若} f(u, v) > 0 \\ +\infty & \text{若} f(u, v) = 0 \end{cases} \]

长度为\(+\infty\)的弧可以从\(W(f)\)中略去。

在网络中寻找关于\(f\)的最小费用增广路，等价于在\(W(f)\)中寻找从\(V_s\)到\(V_t\)的最短路（不能使用Dijkstra，可以使用SPFA或Bellman-Ford）。在原网络中得到相应的增广路\(P\)，沿\(P\)对\(f\)调整。 \[ \alpha = \min\left\{\min_{P+} (c(u, v)-f(u, v)), \min_{P-} f(u, v)\right\} \] \[ f'(u, v) = \begin{cases} f(u, v) + \alpha & (u, v) \in P+ \\ f(u, v) - \alpha & (u, v) \in P- \\ f(u, v) & (u, v) \not\in P \end{cases} \]

如此反复，直到\(f\)为最大流（\(W(f)\)中没有最短路径）为止。

MATLAB实现：

function [f, wf, rwf] = bg(C, vs, vt, r)
% f 最小费用最大流矩阵
% wf 最大流量
% rwf 最小费用
% C 容量矩阵
% vs 源点
% vt 汇点
% r 弧上单位流量的费用
n = length(C);
f = zeros(n);
wf = 0;
rwf = 0;
w = createw(C, r, f);
[path, len] = spfa(w, vs, vt);
while ~isinf(len)
    alpha = Inf;
    for i = 1:length(path)-1
        if C(path(i), path(i+1)) ~= 0
            alpha = min(alpha, C(path(i), path(i+1))-f(path(i), path(i+1)));
        else
            alpha = min(alpha, f(path(i+1), path(i)));
        end
    end
    wf = wf + alpha;
    sumr = 0;
    for i = 1:length(path)-1
        if C(path(i), path(i+1)) ~= 0
            f(path(i), path(i+1)) = f(path(i), path(i+1)) + alpha;
            sumr = sumr + r(path(i), path(i+1));
        else
            f(path(i+1), path(i)) = f(path(i+1), path(i)) - alpha;
            sumr = sumr - r(path(i+1), path(i));
        end
    end
    rwf = rwf + sumr*alpha;
    w = createw(C, r, f);
    [path, len] = spfa(w, vs, vt);
end
end

function w = createw(C, r, f)
% w 伴随网络
% C 容量矩阵
% r 弧上单位流量的费用
% f 流量矩阵
n = length(C);
w = Inf(n);
for u = 1:n
    for v = 1:n
        if f(u, v) < C(u, v)
            w(u, v) = r(u, v);
        end
        if f(u, v) > 0
            w(v, u) = -r(u, v);
        end
    end
end
end

function [path, len] = spfa(G, vs, vt)
% path 最短路径
% len 最短路径长度
% G 网络的邻接矩阵
% vs 源点
% vt 汇点
n = length(G); %
dist = Inf(n, 1);
dist(vs) = 0;
from = zeros(n, 1);
from(vs) = vs;
q = push([], vs);
inq = zeros(n, 1);
inq(vs) = true;
while ~isempty(q)
    [u, q] = pop(q);
    inq(u) = false;
    for v = 1:n
        if dist(v) > dist(u) + G(u, v)
            dist(v) = dist(u) + G(u, v);
            from(v) = u;
            if ~inq(v)
                q = push(q, v);
                inq(v) = true;
            end
        end
    end
end
path = [];
len = 0;
v = vt;
if from(vt) == 0
    len = Inf;
else
    while v ~= vs
        path = push(path, v);
        len = len + G(from(v), v);
        v = from(v);
    end
    path = push(path, vs);
    path = fliplr(path);
end
end

function ansq = push(q, el)
% ansq 压入队尾元素的队列
% q 队列
% el 要压入队尾元素的数字
ansq = [q el];
end

function [v, ansq] = pop(q)
% v q的队首元素
% ansq 去掉队首元素的队列
% q 队列
v = q(1);
ansq = q(2:end);
end
