MATLAB使用入门

发表于 2021-03-20 更新于 2021-10-10 本文字数： 2.7k 阅读时长 ≈ 4 分钟

MATLAB工具箱

工具箱内容位于APP菜单栏内。

MATLAB数值变量

数值变量

MATLAB各种数据类型都以矩阵的形式存在，矩阵是MATLAB最基本的数据对象，并且矩阵的运算都是定义在复数域上的。

数值类型

  • 有符号整型：int8、int16、int32、int64
  • 无符号整型：uint8、uint16、uint32、uint64
  • 单精度浮点型：single
  • 双精度浮点型：double

系统默认的数值类型是double，所以使用double时可以省略double符号。

数值的记述

MATLAB的数只采用习惯的十进制表示，可以带小数点和负号；缺省的数据类型为double。

例如：3、-10、0.001、1.3e10、1.256e-6。

变量命名规则

  • 变量名、函数名对字母的大小写敏感。
  • 变量名的第一个符号必须是英文字母。
  • 变量名可以包含英文字母、下划线和数字。
  • 变量名不能包含空格、标点。
  • 变量名最多可包含63个字符。

冒号的作用

  • 用于生成等间隔的向量，默认间隔为1。例如

    >> A=1:3
A =
     1     2     3

>> B=1:0.5:3
B =
    1.0000    1.5000    2.0000    2.5000    3.0000

  • 用于选出矩阵指定行、列及元素。例如

    >> A=B(i,:)

    表示选出B的第i行元素。

  • 循环语句。例如

    for i=1:3

MATLAB矩阵操作

数组定义

按行（row）和列（column）顺序排列的实数或复数的有序集，被称为数组。

数组中的任何一个数都被称为这个数组的元素，由其所在的行和列标识，这个标识也被称为数组元素的下标或索引。MATLAB将标量视为1×1的数组。

数组的分类

  • 一维数组，也称为向量（vector）。
    • 行向量
    • 列向量
  • 二维数组，也称为矩阵（matrix）。
  • 多维数组

有效矩阵

每行元素的个数必须相同，每列元素的个数也必须相同。

矩阵定义

默认情况下，MATLAB中所有的变量都是双精度矩阵。

要创建一个矩阵变量，只需为其分配值即可，例如2×2矩阵A：

>> A = [1 2; 3 4]
A =
     1     2
     3     4

矩阵基本运算

MATLAB可以简单地对没有特殊语法定义的矩阵进行加减乘除等运算。矩阵必须符合所请求的线性运算的正确大小。如：D = B + C。

使用单引号后缀表示转置，如：A'。

使用^表示矩阵幂运算，如：A^2。

默认情况下，每个变量都属于数值变量。可以使用zeros、ones、eye或rand等函数将矩阵初始化为给定值，分别为0、1、单位矩阵或随机数。

创建二维数组变量

第一种方法：使用方括号“[]”操作符。行与行之间用分号“;”间隔，也可以在分行处用回车键间隔。行内元素用空格或逗号“,”间隔。

>> A2 = [1 2 3;4 5 6;7 8 9]
A2 =
     1     2     3
     4     5     6
     7     8     9

由向量也能构成二维数组。

>> A = [1 2 3]; B = [2 3 4];
>> C = [A;B]
C =
     1     2     3
     2     3     4
>> D = [A B]
D =
     1     2     3     2     3     4
