优化博客（十八）：如何在博客中渲染Mermaid语法

发表于 2020-09-01 更新于 2022-05-15 本文字数： 957 阅读时长 ≈ 2 分钟

Mermaid是Markdown中一个很有用的流程图绘制工具，不过默认Hexo博客并不支持渲染Mermaid语法，NexT主题也默认没有打开渲染功能。因此本文提供两种方法打开Mermaid的渲染功能。

Hexo方法

Hexo的解决方法需要安装一款插件hexo-filter-mermaid-diagrams。

第一步 安装插件

在博客目录打开命令行，执行以下命令：

yarn add hexo-filter-mermaid-diagrams

第二步 编辑配置文件

打开source/_data/next.yml文件，在文件最后加入以下内容：

mermaid:
  enable: true

# 新增styl文件，最终会编译成css代码
custom_file_path:
  style: source/_data/styles.styl

第三步 个性化背景颜色

打开source/_data/styles.styl文件，添加内容：

.mermaid {
    background: transparent;
}

可以把背景颜色改成透明。

NexT方法

设置主题配置文件

打开themes/next/_config.yml文件，找到并修改成下面的部分：

# Mermaid tag
mermaid:
  enable: true
  # Available themes: default | dark | forest | neutral
  theme: default

使用方法

{% mermaid type %}
{% endmermaid %}

比如说

{% mermaid graph TD %}
A[Hard] -->|Text| B(Round)
B --> C{Decision}
C -->|One| D[Result 1]
C -->|Two| E[Result 2]
{% endmermaid %}
